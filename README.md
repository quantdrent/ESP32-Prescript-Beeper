# ESP32-The-Index-Prescript

![License: MIT](https://img.shields.io/badge/License-MIT-green)

> A physical device aiming to recreate the pager device rien used to recive prescripts in canto IX, with wifi feature to send live prescripts

## Required Materials
* **MCU**    : ESP32 C3
* **SCREEN** : 0.91 OLED Screen
* **INPUT**  : 2x Touch capacitive switches (Standard micro switches work but case needs to be tweaked)
> Below are optional -
* **AUDIO**  : DF Player Mini & small speaker
